Showcasing my engineering and automation projects.

Project Overview: Implemented control strategies for automated cruise control and lane changing using a Mercedes AMG C63s model.

Methodology:

  • Designed a P Controller for speed regulation.
  • Developed a PD Controller for lane transitions.
  • Tested performance through simulations.

Findings:

  • P Controller maintains speed but has steady-state errors.
  • PD Controller ensures accurate lane changes.
  • Future improvements include integrating an I component for enhanced accuracy.
