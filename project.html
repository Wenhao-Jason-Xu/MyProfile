Cruise Control & Lane Changing Automation

Project Overview: This project focuses on implementing and validating control strategies for automotive cruise control and lane changing using a Mercedes AMG C63s model.

Methodology:

  • Designed a P Controller for cruise control to regulate speed with error correction.
  • Implemented a PD Controller for lane changing to ensure smooth transitions.
  • Validated performance using simulations under various conditions.

Key Findings:

  • P Controller maintained stability but had steady-state errors under disturbances.
  • PD Controller ensured accurate and safe lane changes.
  • Future improvements include integrating an I component for better accuracy.
