Project Overview: This project explores the design and implementation of automated cruise control and lane-changing strategies using a Mercedes AMG C63s model. The objective was to develop and validate control algorithms that enhance vehicle stability and performance under dynamic driving conditions.

Methodology:

  • Developed a P Controller for speed regulation.
  • Implemented a PD Controller for lane transitions to ensure smooth changes.
  • Simulated system performance under real-world conditions.

Findings:

  • P Controller effectively maintained speed but exhibited steady-state errors.
  • PD Controller improved lane change accuracy, reducing drift.
  • Future work involves adding an I component for enhanced precision.
